My Projects

Statistical Calculator

I developed a Python application with a Tkinter-based Graphical User Interface (GUI) designed to calculate essential measures of central tendency and dispersion. This user-friendly tool allows users to input a list of numbers and instantly obtain statistical results such as mean, median, mode, standard deviation, and more. By leveraging Python's simplicity and Tkinter's interactive capabilities, I crafted an intuitive interface that streamlines basic statistical calculations. This project not only showcases my proficiency in Python programming but also my ability to create practical solutions with a focus on usability and accessibility.
